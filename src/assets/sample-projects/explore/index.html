This sample project contains a series of one page html files developed by the community for the One Html Page Challenge. Explore and play with these HTML files on the files panel in Phoenix. Please join the challenge if you wish to by following the below links.

The goal is to create anything you want within 1 single html file. Practice your skills with no assistance from libraries, no separation of files, and no assistance of a modern framework. How creative can you be with such restrictions?
